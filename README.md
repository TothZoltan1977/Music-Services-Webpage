# Music Website Project

This is a modern, responsive music website that showcases different music genres, services, and unique visual elements.  

## Features

- Responsive header (navbar) with hamburger menu below 600px
- Animated search bar
- Services and music works presentation with images and descriptions
- Interactive image and SVG slider
- Auto-changing image gallery (slider)
- Stylish footer with contact info and social media icons

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (animations, slider, search bar)
- SVG images
